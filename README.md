# AP Textile Industries — Website Project Submission

This repository contains the finalized website for **AP Textile Industries**, a premium textile wholesaler and exporter based in Panipat, Haryana, India.

## 📂 Project Structure

- **`NextJS-Version/`**: The modern web application built with **Next.js 16**, TypeScript, and Tailwind CSS. This version includes a fully functional contact form integrated with the **Resend API**.
- **`Vanilla-HTML-Version/`**: A syllabus-compliant version built using **Vanilla HTML, CSS, and JavaScript**. It provides the same premium aesthetic using the Tailwind CDN for zero-build-step accessibility.
- **`write-up.md`**: The official project report containing business goals, target audience analysis, design decisions, and technology stack overview.

## 🚀 Key Features

- **Dual-Stack Implementation**: High-performance React version + light-weight static HTML version.
- **Live Inquiry System**: Contact form wired to `aptextileindustries1@gmail.com` via Resend Node SDK.
- **Premium Design**: Sophisticated typography (Playfair Display) and a navy-gold color palette.
- **Comprehensive Product Catalog**: Detailed sections for Cotton Yarn, Polyester Yarn, Malai Dori, Bath Towels, and Bath Mats.
